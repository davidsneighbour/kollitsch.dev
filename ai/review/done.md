# Completed or retired opportunities

Move finished, closed, intentionally retired, or superseded items here.

The AI must read this file before re-adding an item to `ToDo.md`.

## Format

Use one section per completed or retired item.

### `topic-slug`

* Final status:
* Origin:
* Closed on:
* Reason:
* Follow-up:
* Notes:

## Example

### `docs-old-install-guide-cleanup`

* Final status: done
* Origin: ai
* Closed on: 2026-03-22
* Reason: Replaced with the new setup guide in `README.md`.
* Follow-up: none
* Notes: Re-add only if the old guidance reappears.
